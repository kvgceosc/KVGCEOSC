Git in Action — Workshop, Website & T-Shirt Launch

The KVG College of Engineering Open Source Community (KVGCEOSC) organized an engaging workshop titled “Git in Action” on May 15, 2025, at 2:00 PM in the Smart Classroom (406). The event aimed to provide students with a hands-on understanding of Git and GitHub — essential tools for modern software development and collaboration.


Dignitaries Present

The event was graced by esteemed guests: Dr. Suresha V, Principal, KVG College of Engineering and Prof. Ujwal U J, Head of the Department, Computer Science & Engineering. Both dignitaries inaugurated the event and the website launch. The faculty coordinators were also present.


Workshop Overview: “Git in Action”

The technical session, “Git in Action,” focused on understanding the fundamental and practical aspects of Git — a version control system widely used by developers around the world. Topics covered included repository setup, basic Git commands (clone, commit, push, pull, branch), collaboration workflows, and a hands-on demonstration by Likith K R and Pavan M.


Website and T-Shirt Launch

As part of the event, KVGCEOSC proudly launched its official website and club T-shirt. The website strengthens the club’s digital presence, and the T-shirt — with the tagline “Aspire. Build!” — symbolizes the spirit of innovation and unity among members.


Conclusion

The “Git in Action” workshop was an inspiring and informative event that blended technical learning with community celebration. Students gained valuable insights into real-world version control systems, while the launches added excitement and motivation. The KVGCEOSC team expressed gratitude to the principal, HOD, faculty coordinators, and all participants for making the event a success.
